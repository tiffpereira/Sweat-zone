# Sweat-zone

## Description
For people that are starting their fitness goals or are pros at training, this site provides a guide for all the best workouts. It will include upper and lower body strength training and include a description of the workout, time frame, and youtube link. Users will also be able to submit their own workouts to be featured on the page. 

The following technologies will be used: Mongoose, express, node, react, and CSS. 


## Backend 
The database will contain a title, brief description, estimated time frame, and a youtube link.
